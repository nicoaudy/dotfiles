## My personal dotfiles config 🔥

### Basic Requirements

- ctags
- php-cs-fixer

```bash
brew install ctags
brew install php-cs-fixer
```

### Installation

```bash
https://github.com/nicoaudy/dotfiles.git
```

Don't forget to add +x to *.sh.

```bash
cd dotfiles
cd scripts
chmod +x vim-install.sh
chmod +x vim-save.sh
```

### Others
- iterm scheme (arthur)


Save changes vim from machine using vim-save script. For install to new machine using vim-install script.
